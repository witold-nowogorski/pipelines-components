# Automl

This subcategory contains components in the **Automl** group:

- [Autogluon Models Training](./autogluon_models_training/README.md): Train AutoGluon models, select the top N, and refit each on the full dataset.
- [Autogluon Timeseries Models Full Refit](./autogluon_timeseries_models_full_refit/README.md): Refit a single AutoGluon timeseries model on full training data.
- [Autogluon Timeseries Models Selection](./autogluon_timeseries_models_selection/README.md): Train and select top N AutoGluon timeseries models based on leaderboard.
- [Leaderboard Evaluation](./autogluon_leaderboard_evaluation/README.md): Evaluate refitted AutoGluon models and generate a leaderboard.
- [Timeseries Leaderboard Evaluation](./autogluon_timeseries_leaderboard_evaluation/README.md): Evaluate refitted AutoGluon timeseries models and generate a leaderboard.
